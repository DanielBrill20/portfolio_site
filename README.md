# Daniel Brill | Creative Technologist Portfolio

A single-page portfolio with individual project detail pages. Built with vanilla HTML, CSS, and a small JS nav toggle. Live-hosted via GitHub Pages.

## Live Site
- GitHub Pages: https://danielbrill20.github.io/portfolio_site/

## Structure
- `index.html` — landing page (hero, bio, projects, art, contact)
- `*.html` — project detail pages (Spotify Pixel Art Display, Sumobot, ESP32 builds, etc.)
- `css/styles.css` — global styles and responsive breakpoints
- `js/main.js` — mobile nav hamburger toggle
- `assets/` — images and logo assets

## Deployment (GitHub Pages)
- Branch: `main` (static hosting)
- Source: `/(root)`
- After updates push to `main`, Pages will rebuild automatically
